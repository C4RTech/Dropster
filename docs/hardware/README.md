1. Coloque las fotos del dispositivo en esta carpeta con los nombres exactos especificados.
2. Genere el manual de usuario en PDF y colóquelo aquí.
3. Prepare los documentos técnicos según las especificaciones y agréguelos.
4. Una vez agregados todos los archivos, el README principal del repositorio mostrará automáticamente las referencias correctas.

## Especificaciones del Dispositivo

Para referencia rápida:

- **Microcontrolador**: ESP32 WROVER 32D Dev Kit V3
- **Sensores**: BME280, SHT30, PZEM004T, ultrasónico
- **Actuadores**: Ventiladores, compresor, bombas
- **Pantalla**: TFT ILI9341 2.8" táctil
- **Alimentación**: 110V AC
- **Dimensiones**: 60cm x 40cm x 120cm
- **Capacidad**: 20 litros
- **Producción**: Hasta 15 litros/día

Esta documentación asegura que el repositorio proporcione información completa tanto sobre la aplicación móvil como sobre el dispositivo hardware.
